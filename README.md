# 🚧 Road Damage Management System

A web-based application designed to report, track, and manage road damages efficiently. This system helps citizens report road issues and allows authorities to monitor and resolve them in a structured manner.

---

## 📌 Project Overview

The **Road Damage Management System** is developed using Java-based web technologies. It enables users to register, log in, and report road damages with details and images. The system stores complaints in a database and allows viewing and management of reported issues.

---

## 🚀 Features

* 🔐 User Registration & Login Authentication
* 📝 Report Road Damage with Description & Image
* 📂 View Submitted Complaints
* 📊 User Dashboard
* 🔄 Logout Functionality
* 💾 Database Integration for Data Storage

## ⚙️ How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/your-username/Road_Damage_Management_System.git
   ```

2. Import into Eclipse as **Dynamic Web Project**

3. Configure Apache Tomcat Server

4. Setup PostgreSQL Database:

   * Create database
   * Update DB credentials in `DBConnection.java`

5. Run the project on server
